Project Overview

Project Details

I called this the Mean Stack Blog because I created a blog-like site that was built using the MEAN stack. I used local authorization using MongoDB and express.js/node.js so you can login/logout. The built in text editor and views are created in AngularJS and you can save your posts to the Mongo data base. Posts can be retrieved by clicking on the History link within your account.
